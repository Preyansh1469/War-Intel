# War-Intel

## Project Overview
War-Intel is a comprehensive intelligence gathering platform designed to provide insights into military activities, geopolitical issues, and conflict zones. It helps users analyze various aspects of warfare and conflicts around the world.

## Features
- **Real-time data collection**: Aggregates data from various sources including news articles, social media, and government reports.
- **Visual analytics**: Provides graphical representations of data to help users understand trends and patterns.
- **Geolocation mapping**: Displays conflict zones on interactive maps for better situational awareness.
- **Custom alerts**: Notifies users about significant events based on their interests.
- **User-friendly interface**: Designed for ease of use to allow non-experts to navigate and access insights.

## Tech Stack
- **Frontend**: React, Redux, Bootstrap
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **APIs**: RESTful APIs for communication between frontend and backend
- **Deployment**: Docker, AWS

## Setup Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/Preyansh1469/War-Intel.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd War-Intel
   ```

3. **Install dependencies**
   - For the frontend:
     ```bash
     cd client
     npm install
     ```
   - For the backend:
     ```bash
     cd server
     npm install
     ```

4. **Run the application**
   - For the backend:
     ```bash
     cd server
     npm start
     ```
   - For the frontend:
     ```bash
     cd client
     npm start
     ```

5. **Access the application**: Open your web browser and go to `http://localhost:3000`.
